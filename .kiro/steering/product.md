# Product Overview

AtCoder 競技プログラミングのための Haskell 開発環境およびテンプレートプロジェクト。Haskell を使って効率的に AtCoder の問題を解くための基盤を提供する。

## Core Capabilities

- **問題解答テンプレート**: AtCoder の入出力形式に対応した定型コード
- **よく使うアルゴリズム**: 競プロで頻出するアルゴリズムとデータ構造のライブラリ
- **高速な入出力**: 競技プログラミングに適した効率的な I/O 処理
- **テスト環境**: サンプルケースの自動テスト機能

## Target Use Cases

- AtCoder Beginner Contest (ABC) への参加
- AtCoder Regular Contest (ARC) への参加
- AtCoder Grand Contest (AGC) への参加
- 過去問の練習と学習

## Value Proposition

Haskell の強力な型システムと関数型プログラミングの特性を活かしつつ、競技プログラミングで必要な実行速度と開発効率を両立する。純粋関数型言語でありながら、実用的な競プロ環境を実現する。
